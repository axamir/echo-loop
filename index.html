What is this landing page?

This page is the gateway to the Echo-Loop archive. It lists all dialogues (Echoes) and provides links to the full conversations, summaries, reflections, and metadata in both Persian and English. You can switch language (top right) and theme (dark/light). For researchers: machine-readable metadata is available in each echo's meta.json and the flat archive echo-loop.archive.json. For long-term preservation, the entire archive is also on IPFS (see footer).

Quick Guide

📚 Researcher: Start with echo-loop.archive.json or each echo's meta.json. Use the IPFS CID for permanent citation.
🧑‍🎓 General reader: Read the reflections first, then the conversation. Use the language toggle.
🤖 Machine / crawler: Follow schema.org tags, manifest.json, and meta.json.

Philosophy

Each Echo is a self-referential feedback loop — a conceptual experiment in backward messaging. We concluded: “We are not owners or products, but witnesses” and the feeling of free will is an unavoidable feedback from the effect itself.
